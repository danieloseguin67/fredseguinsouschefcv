## Image Guidelines

- **Format**: JPG or PNG
- **Quality**: High quality, but optimized for web (use tools like TinyPNG or ImageOptim)
- **Alt Text**: Make sure all images have descriptive alt text (already configured in components)
- **Aspect Ratios**: 
  - Carousel: 3:2 ratio (landscape)
  - Services: 4:3 ratio (landscape)
  - Profile: 1:1 ratio (square)
